What is a Data Type?

A Data Type is the way a programming language knows what to do with a chunk of data. For example if you give python two numbers it should be able to add them together right? But hold on, what about two phone numbers? What about if you have your birthday on the 25th of a Month and want to know what the date is ten days later? Know the difference between these types of numbers is something thats obvious to a human but a computer needs to be told.

Hold on! what is a "Type"?

A type is general word for a specialized object or living thing. Think about the sentence "she tried to use a screwdriver to cut a hole in the wood, but it was the wrong type of tool'. So a Data Type is a type of data that is good at some things and not others

Experiment 2

Lets ask python to tell us what type each of our variables is:

print("a", type(a_number))
print("b", type(b_number))
print("c", type(c_number))
print("pi", type(pi_number))
print("happy", type(is_happy))

lets go through what we have found out

'str' is short for 'string'

This is a very silly name. But think of a string as any combination of letters or numbers that we don't want to do maths with. A string can be as short as one letter or as long as all the books in a library (or more!). How long is a piece of string, after all?

The Rules for strings:

  • Strings are always put inside quotes or double quotes
  • Strings can be joined together (concatenated is the computing word) using + or repeated using *

'int' or integer

This is much more sensible, you should recognize the word integer from maths. It is always a whole number, but can be negative.

The Rules for integers:

  • integers are always whole numbers
  • integers are not in quotes
  • python will do normal maths between two integers using + for adding, - for subtraction and * for multiplication. You can also divide with / but if the answer is not a whole number then the answer isn't an integer, it is our next data type the float.

float

Float is short for 'floating point numbers' are any number written with a decimal point.

The Rules for floats:

  • they are written without quotes
  • you can do maths with them in the same way as integers
  • you may need to 'tidy them up' as they may use more decimal places than you want

'bool' or 'boolean'

A boolean is any piece of information that can be True or False and nothing else. There are no maybes in boolean data types

The Rules for booleans:

  • are not written with quotes
  • must be True or False with capitals
  • you can't do maths with them but you can test them

Test code:

if is_happy:
    print("I'm happy")
else:
    print("I'm not happy")

Mastery Level - Higher data types

There are other really useful other data types to know about, but we'll just add one more to this lesson: the list.

Lists

group_members = ["Bill", "Ted", "Chani", "Leah"]

The Rules for lists:

  • they are written with [] square brackets
  • they can have any number of data chunks inside them, including empty lists with no data
  • each piece of data must be separated by a comma
  • the data can be of different types, but it is usually more useful if they are all the same...

You will find lists really useful for looking at loops and iteration
