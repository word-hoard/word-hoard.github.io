4 Truth

What is True?

Boolean Data Types can only be True or False, but that doesn't really mean much until we know what is true or false. Try and name your Booleans so they give you a clue as to what this is is about: for example is_a_fruit, is_paid_for, is_group_member.

This is OK if you already know the truth about something, but you often need your code to decide what is True for you.

Exercise 1

Try this code and run it a few times, entering in different names each time, including 'bob' and 'Bob'

name =  input("what is your name? ")

is_bob = name == "bob"

print(f"{name=}  {is_bob=}")

This bit of code is a bit confusing though - what is going on the second line with the = sign and the == sign?!?

The quick answer is the double = is a comparison operator. You have already met arithmetic operators (the maths symbols +-*/, oh and %, **, // as well...) and the assignment operator =.

Basically comparison operators compare two things like this:

  • == is equal to 3 == 3
  • != is not equal to 3 != 4
  • > is greater than 10 > 5
  • < is less than 3 < 5
  • >= is greater than or equal to 10 >= 10
  • <= is less than or equal to 4 <= 14

test_numbers = [3, 7, 33, 1, 740]

IF

Often we don't bother storing our Boolean truth as a variable, but use it immediately to perform a selection algorithm like this:

score = 45

if score > 50:
    print("excellent score")
else:
    print("better luck next time...")

If statments as selection algorithm can get quite complicate, but try as basic one like this before going back to iteration
